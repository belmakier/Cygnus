Detailed Description

Determine spectroscopic information from matrix elements.

From the matrix elements and energies defined in the Nucleus class this class determines transition rates, lifetimes, mixing ratios and branching ratios. Specifically, this is for comparison with input experimental (literature) values in the minimization process.
