Detailed Description

Define an experimental range, for determining integrated yields.

The ExperimentRange class uses Integral to calculate the cross- section meshpoints, state-by-state. It then fits the theta-energy cross-section dependence and integrates the surface, along with a correction for energy loss in the target, in order to calculate an integrated cross section.
