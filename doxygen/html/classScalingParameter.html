Detailed Description

Define common scaling parameters for calculation to experimental data. 

Due to a typical lack of absolute scaling information Coulomb excitation

fits require some degree of scaling to reproduce the data. This class defines these scaling parameters in a sensible way. 

A single scaling parameter can be coupled to one or more experiments,

identified in the experimentNumber vector, containing the indices of the relevant experiments. 

Upper and lower limits can also be defined for the purposes of aiding

the fit.
