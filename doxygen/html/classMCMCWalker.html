Detailed Description

Markov-Chain Monte Carlo iterator.

MCMCWalker

Using a Metropolis-Hastings algorithm the MCMCWalker class iterates through the parameter space, with a weighting towards the most likely solutions, causing eventual convergence while sampling the probability distribution of the parameters.
