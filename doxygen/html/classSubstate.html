Detailed Description

Holder class for information on magnetic substates including M,

connection information, etc.

Excitation calculations are performed substate-by-substate, so this class (rather than the State class) contains the majority of the important coupling information.

Contained within this class is a list of connections to other substates, defined by the Connections class.

Also included is an index for the State object, of which the substate is a... substate.
